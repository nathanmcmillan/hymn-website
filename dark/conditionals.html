Conditionals

Conditionals are used to control the flow of a program.

if 1 < 2 {
  print("Yes!")
}

if 1 < 2 {
  print("Yes!")
} elif 2 < 4 {
  print("Yes!")
}

if 1 < 2 {
  print("Yes!")
} elif 2 < 4 {
  print("Yes!")
} else {
  print("No!")
}
