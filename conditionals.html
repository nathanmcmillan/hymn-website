Conditionals

Conditionals are used to control the flow of a program.

if 1 < 2 {
  echo "True!"
}

if 1 < 2 {
  echo "Foo"
} elif 2 < 4 {
  echo "Bar"
}

if 1 < 2 {
  echo "Yes!"
} elif 2 < 4 {
  echo "..."
} else {
  echo "No!"
}
